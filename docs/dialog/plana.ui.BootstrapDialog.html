decorateInternal(element)

Decorates the element for the UI component. If the element is in the document, the enterDocument method will be called. Any buttons *must* be inside a 'modal-footer' element, otherwise they are not added to the set of handled buttons. A cancel button must have 'data-role' == 'cancel' and the default button must use the css class 'btn-primary'
